Project 4 : Daily Expense Tracker

Abstract :

The Daily Expense Tracker is a simple yet efficient command-line application designed to help users manage their daily expenses. This project allows users to register and securely log in before accessing its core features. Once logged in, users can add and view detailed records of their expenditures by specifying the category, description, amount, and date of each expense. The application follows a menu-driven interface that guides users through the registration, login, and expense management processes. By maintaining a structured format for storing and displaying expenses, the tool promotes better financial awareness and budgeting habits. This project is ideal for individuals who want to keep track of their spending in a user-friendly and secure manner. It demonstrates foundational programming concepts such as user authentication, data input/output handling, and menu-based navigation.
